📚 Component Storybook

フロントエンドコンポーネントのストーリーとドキュメント

🏠 メインアプリケーション
📊 MetricsCards
売上、顧客数、満足度などの主要メトリクスを表示するカードコンポーネント

Stories: Default, HighValues, LowValues, NoData

Props: metrics: Metrics | null

📈 Charts
売上推移、カテゴリ別構成、地域別売上、年齢分布を表示するチャートコンポーネント

Stories: Default, HighVolumeData, MinimalData, EmptyData, NoData

Props: chartData: ChartData | null

🎛️ FilterPanel
期間、カテゴリ、地域、売上範囲、年齢などのフィルター設定パネル

Stories: Default, WithLimitedOptions, WithExtensiveOptions, EmptyOptions

Props: DataServiceを通じてフィルターオプションを取得

📋 DataTables
売上データと顧客データをタブ形式で表示するテーブルコンポーネント

Stories: Default, LargeDataset, SmallDataset, HighSatisfactionData, LowSatisfactionData, EmptyData, NullData

Props: salesData: SalesData[] | null, customerData: CustomerData[] | null

🎯 MatrixView
カテゴリ×地域、年齢層×満足度、月別×カテゴリのクロス集計表示

Stories: Default, SmallDataset, SingleCategoryRegion, HighSatisfactionCustomers, LowSatisfactionCustomers, WideAgeRange, EmptyData, NullData

Props: salesData: SalesData[] | null, customerData: CustomerData[] | null, chartData: ChartData | null

🛠️ 開発環境

正式なStorybookサーバーを起動するには：

npm run storybook

📝 使用方法

各コンポーネントには複数のストーリーが定義されており、異なるプロパティ値でのコンポーネントの動作を確認できます。
