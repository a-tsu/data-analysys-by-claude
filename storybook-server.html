📚 Component Storybook

各コンポーネントのストーリーとサンプルデータを確認できます。

🏠 メインアプリケーション
📊 MetricsCards
売上、顧客数、満足度などの主要メトリクスを表示するカードコンポーネント
デフォルト 高い値 低い値 データなし
📈 Charts
売上推移、カテゴリ別構成、地域別売上、年齢分布を表示するチャートコンポーネント
デフォルト 大量データ 最小データ 空データ
🎛️ FilterPanel
期間、カテゴリ、地域、売上範囲、年齢などのフィルター設定パネル
デフォルト 限定オプション 豊富なオプション 空オプション
📋 DataTables
売上データと顧客データをタブ形式で表示するテーブルコンポーネント
デフォルト 大量データ 少量データ 高満足度データ 低満足度データ
🎯 MatrixView
カテゴリ×地域、年齢層×満足度、月別×カテゴリのクロス集計表示
デフォルト 小規模データ 高満足度顧客 幅広い年齢層
